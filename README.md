# C++ Bug Fix Challenge

A fun and interactive web-based game designed to test and improve your C++ debugging skills. Fix common C++ bugs under a time limit and challenge yourself to achieve the highest score!

## Demo

Experience the game in action!


## Features

-   **Interactive Challenges**: Solve various C++ code snippets with common bugs.
-   **Time Limit**: Each challenge has a timer, adding excitement and urgency.
-   **Scoring System**: Track your progress and aim for a perfect score.
-   **Immediate Feedback**: Get instant notifications if your fix is correct or if time runs out.
-   **Clean UI**: A sleek, developer-friendly interface designed for focus.
-   **Responsive Design**: Play on various devices, from desktops to mobile phones.

## How to Play

1.  **Start the Game**: Click the "Start Game" button on the welcome screen.
2.  **Analyze the Buggy Code**: Read the challenge description and examine the provided C++ code in the "Buggy Code" section.
3.  **Fix the Code**: Type or paste your corrected C++ code into the "Your Fixed Code" textarea.
4.  **Submit Your Answer**: Click "Submit Answer" to check your solution.
    *   If correct, you'll see a success screen and proceed to the next challenge.
    *   If incorrect, you'll receive a hint to try again.
5.  **Time's Up**: If you run out of time, the correct answer will be displayed, and you can move to the next challenge.
6.  **Skip Challenge**: If you're stuck, you can skip the current challenge, which will reveal the correct answer and move you forward.
7.  **Game Over**: The game ends when you complete all challenges or run out of time on a challenge. Your final score will be displayed.

## Technology Stack

-   **HTML5**: For the basic structure and content of the web page.
-   **CSS3**: For styling the game, including responsive design and animations.
-   **JavaScript**: For all game logic, challenge management, timer, scoring, and UI interactions.
